## 🆘 Troubleshooting

### If GitHub Pages doesn't work:
1. Go to: https://github.com/Shme-CS/js-quiz-app/settings/pages
2. Make sure "Source" is set to "GitHub Actions"
3. Check the Actions tab for any errors
4. Wait a few more minutes (first deployment can take 5 minutes)
