## Enable GitHub Pages

**Manual Setup** (1 minute):

1. Go to: [Repository Settings → Pages](https://github.com/Shme-CS/js-quiz-app/settings/pages)
2. Under "Source":
   - Branch: **main**
   - Folder: **/ (root)**
3. Click **Save**
4. Wait 2 minutes
5. Visit: https://shme-cs.github.io/js-quiz-app/

## 📝 After Deployment

Once you deploy to Vercel or Netlify, update your README.md:

1. Copy your deployment URL
2. Open `README.md`
3. Replace the placeholder with your actual URL
4. Commit and push:
   ```bash
   git add README.md
   git commit -m "Add deployment URL"
   git push
   ```
